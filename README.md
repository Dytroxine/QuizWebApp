Developed a real-time online quiz platform where users compete to earn promo codes as rewards. The project is built with Django for the backend and Aiogram for Telegram integration, providing an interactive user experience both on the web and through messaging.

Key Features:

Real-time gameplay: Online quizzes are powered by WebSockets, enabling instant updates and smooth interactions between participants.

Admin panel: Fully-featured Django admin interface for creating and managing quizzes, questions, and promo codes with ease.

User interface: Authentic and intuitive design focused on usability and engagement.

Database: PostgreSQL for reliable storage of quizzes, users, and game results.

Telegram integration: Aiogram-based bot allows users to participate via Telegram, enhancing accessibility and engagement.

Reward system: Winners automatically receive promo codes, linking the platform to marketing campaigns or partner services.
